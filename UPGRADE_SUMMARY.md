## 📋 完成内容

### 1. 模拟硬件系统 ✓

**文件**: `backend/simulate_vehicle.py`

**功能**:
- 支持多车辆模拟（默认3个）
- 合理的运输路线：北京 → 天津（约120km）
- 15秒上报间隔
- 80 km/h速度模拟
- 电池消耗模拟
- 信号强度变化模拟

**使用方式**:
```bash
# 启动3个车辆
python simulate_vehicle.py multi 3

# 启动单个车辆
python simulate_vehicle.py vehicle_001
```

## 🔧 自定义配置

### 修改车辆数量
```bash
python simulate_vehicle.py multi 5  # 启动5个车辆
```

### 修改上报间隔
编辑 `backend/simulate_vehicle.py`:
```python
self.interval = 10  # 改为10秒
```

### 修改速度
编辑 `backend/simulate_vehicle.py`:
```python
self.speed_kmh = 100  # 改为100 km/h
```

### 修改路线
编辑 `backend/simulate_vehicle.py` 中的 `generate_route()` 方法
